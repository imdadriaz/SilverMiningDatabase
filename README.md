# Silver Mining Database

This repo stores the SQL schema and starter data for the Silver Mining project.

## Files
- `db/SilverMiningDatabase.sql`, main SQL script
- `docs/setup.md`, setup instructions

## Requirements
- MySQL 8.0 or compatible
- MySQL Workbench or MySQL CLI

## Run
Execute `db/SilverMiningDatabase.sql`.

The script:
- creates the database
- creates all tables
- inserts starter data
